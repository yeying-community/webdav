# warehouse
Browser asset repository

## Node version
```sh
Node.js >= 18.x
```

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```


### The packaging path points to

```sh
构建后的文件将生成在：./dist/
```
